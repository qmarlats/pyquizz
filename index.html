Dépendences

PyQuizz utilise simpleaudio pour lire les musiques, il faut donc l'installer (pip install simpleaudio) ainsi que ses dépendences (voir ici) :

  • Python3 et ses paquets "dev" (Python 3.5+ est requis)
  • ALSA et ses paquets "dev"

Limitations

Les limitations sont les mêmes que simpleaudio (see here), PyQuizz ne peut donc que lire :

  • fichiers WAV
  • fichiers monos ou stéréos
  • 8, 11.025, 16, 22.05, 32, 44.1, 48, 88.2, 96, ou 192 kHz

Installation

Installation sur une dérivée de Debian :

  • Installer les dépendences de simpleaudio : sudo apt install -y python3-dev libasound2-dev
  • Installer python3-pip: sudo apt install python3-pip
  • Il vaut mieux utiliser un environnement virtuel :
    • Utiliser l'existant : source env/bin/activate
    • Si cela ne fonctionne pas, en créer un :
      • Créer l'environnement : virtualenv my_env
      • L'utiliser : source my_env/bin/activate
      • Installer les modules Python requis : pip3 install simpleaudio wave sphinx peewee

Utilisation

Entrer dans l'environnement virtuel : source env/bin/activate. Puis aller dans le dossier pyquizz et exécuter pyquizz.py avec Python3 (python3 pyquizz.py), puis jouer.
